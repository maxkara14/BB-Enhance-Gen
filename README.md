**✨ BB Enhance Generation**

Панель кнопок для генерации и редактуры текста для SillyTavern. Расширение добавляет симпатичные кнопки прямо над полем ввода сообщения.

*Доступные функции:*

Кнопки можно включать и отключать по вашему желанию в стандартном меню расширений SillyTavern.

* ✨ **Enhance:** Превращает ваш черновик в кинематографичный, детализированный текст. ИИ добавляет чувственные описания (звуки, запахи, текстуры) и внутренние мысли персонажа, делая текст объемнее, но не двигая сюжет вперед.
* 🔮 **Improve:** Работает как профессиональный строгий редактор. Исправляет ошибки, заменяет слабые глаголы на изящные литературные аналоги, сглаживает текст, полностью сохраняя вашу оригинальную длину, структуру и смысл. Никакой отсебятины и ролевой игры — только полировка текста.
* 🎲 **Random Event:** Читает контекст последних сообщений и генерирует неожиданный сюжетный поворот, создавая напряжение и подталкивая историю в новом направлении.

---

⚙️ Установка расширения:

1. Откройте SillyTavern.
2. Перейдите во вкладку Расширения (Extensions) -> Установить расширение (Install extension).
3. Вставьте ссылку на этот GitHub репозиторий: `https://github.com/maxkara14/BB-Enhance-Gen`
4. Перезагрузите страницу (F5).
5. Панель с кнопками появится прямо над полем ввода текста.
