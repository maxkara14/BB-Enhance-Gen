**🎬 BB Enhance Generation**

Панель кнопок для генерации, редактуры текста и управления сюжетом в SillyTavern. Превращает текстовую ролевую в интерактивную игру.

* ✨ **Enhance:** Нейросеть берет ваш короткий набросок и расширяет его в детализированный, литературный текст, добавляя описания чувств и окружения.
* 🔮 **Improve:** Строгий редактор. Полирует ваш черновик, исправляя ошибки и улучшая слог, без изменения изначального объема и смысла.
* 🎬 **Event Director (Режиссер):** Всплывающее меню для создания неожиданных поворотов. Позволяет направить сюжет (Disaster, Blessing, Tension, Absurd, Time Skip) для вас или для бота.
* 🎲 **Action Roll:** Механика проверки навыков. Анализирует действие, бросает на экране виртуальный 3D-кубик (d20) и заставляет ИИ отыграть ваш триумф или жестокое фиаско в зависимости от результата.
* 📍 **Fast Travel:** Система быстрого перемещения. Оценивает сцену и предлагает 3 логичных варианта (или случайный энкаунтер) для смены локации. Блокируется, если персонаж занят важным диалогом или боем.

---

⚙️ Установка расширения:

1. Откройте SillyTavern.
2. Перейдите во вкладку Расширения -> Установить расширение.
3. Вставьте ссылку на этот GitHub репозиторий: `https://github.com/maxkara14/BB-Enhance-Gen`
4. Перезагрузите страницу (F5).
5. Панель с кнопками появится слева от «волшебной палочки» в буковке Е.
