**✨ BB Enhance Generation**

Умная панель кнопок для генерации и редактуры текста для SillyTavern. Расширение добавляет симпатичные кнопки прямо над полем ввода сообщения.

*Доступные функции:*

Кнопки можно включать и отключать по вашему желанию в стандартном меню расширений SillyTavern.

* ✨ **Enhance:** Превращает ваш сухой черновик в кинематографичный, детализированный текст. ИИ добавляет чувственные описания (звуки, запахи, текстуры) и внутренние мысли персонажа, делая текст объемнее, но не двигая сюжет вперед. Идеально подстраивается под характер вашей персоны.
* 🔮 **Improve:** Работает как профессиональный строгий редактор. Исправляет ошибки, заменяет слабые глаголы на изящные литературные аналоги, сглаживает текст, полностью сохраняя вашу оригинальную длину, структуру и смысл. Никакой отсебятины и ролевой игры — только полировка текста.
* 🎬 **Event Director:** AI-режиссер, пришедший на смену случайным событиям. При нажатии открывается всплывающее меню, где вы можете выбрать атмосферу случайного события: 💥 Опасность (Disaster), 🎁 Удача (Blessing), ❤️ Напряжение (Tension) или 🃏 Комедия (Absurd).
* **Два режима работы:** Вы можете сгенерировать событие **«Мне»** (текст появится в вашем поле ввода) или применить его **«Боту»** (скрипт заставит ИИ переписать свое последнее сообщение, органично вплетая выбранное событие прямо в свой ответ.)

---

⚙️ Установка расширения:

1. Откройте SillyTavern.
2. Перейдите во вкладку Расширения -> Установить расширение.
3. Вставьте ссылку на этот GitHub репозиторий: `https://github.com/maxkara14/BB-Enhance-Gen`
4. Перезагрузите страницу (F5).
5. Панель с кнопками появится прямо над полем ввода текста.
